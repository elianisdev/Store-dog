Media Query

Media Query é uma técnica de CSS3 que permite adaptar o layout de uma página web a diferentes dispositivos, como smartphones, tablets, notebooks e desktops.

Para isso, utilizamos a regra @media, que permite aplicar estilos diferentes de acordo com as características do dispositivo, como largura da tela, altura da tela, orientação, resolução, entre outros.

Por exemplo, podemos criar um layout específico para smartphones, outro para tablets e outro para desktops.
